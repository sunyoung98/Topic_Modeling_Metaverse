'앗, 이런!' 페이지 비정상 종료 및 기타 페이지 로드 오류 문제 해결하기

웹페이지 대신 '앗, 이런!' 오류가 표시된다면 Chrome에 로드 문제가 있는 것입니다.

페이지 로드 오류 코드 및 문제

다음 오류 코드는 페이지를 여는 중에 문제가 발생했다는 의미입니다.

  • '앗, 이런!': Chrome에서 페이지를 로드하는 데 문제가 있습니다.
  • ERR_NAME_NOT_RESOLVED: 호스트 이름(웹 주소)이 존재하지 않습니다.
  • ERR_INTERNET_DISCONNECTED: 기기가 인터넷에 연결되지 않았습니다.
  • ERR_CONNECTION_TIMED_OUT 또는 ERR_TIMED_OUT: 페이지에 연결하는 데 시간이 너무 오래 걸립니다. 인터넷 연결이 너무 느리거나 웹페이지에 접속한 사용자가 많을 수 있습니다.
  • ERR_CONNECTION_RESET: 웹페이지 연결을 방해하는 요소가 발생했습니다.
  • ERR_NETWORK_CHANGED: 페이지를 로드하는 중에 기기의 네트워크 연결이 해제되었거나 새로운 네트워크에 연결되었습니다.
  • ERR_CONNECTION_REFUSED: 웹페이지에서 Chrome의 연결을 허용하지 않았습니다.
  • ERR_CACHE_MISS: 웹페이지에서 이전에 입력한 정보를 다시 한 번 제출하도록 요청합니다.
  • ERR_EMPTY_RESPONSE: 웹사이트에서 데이터를 전혀 전송하지 않았으며 다운되었을 수 있습니다.
  • ERR_SSL_PROTOCOL_ERROR: 페이지에서 전송된 데이터를 Chrome에서 이해하지 못했습니다.
  • ERR_BAD_SSL_CLIENT_AUTH_CERT: 클라이언트 인증서에 오류가 발생하여 웹사이트(예: 은행 또는 회사 웹사이트)에 로그인할 수 없습니다.

전체 오류 코드 목록은 chrome://network-errors/에서 확인하세요.

다음 중 하나 이상의 문제가 발생할 수 있습니다.

  • 사이트에 연결할 수 없습니다.
  • 웹사이트가 열리지 않습니다.
  • HTTPS 사이트가 열리지 않습니다.
  • 사진이 로드되지 않습니다.
  • 새 탭이 로드되지 않습니다.

문제를 해결하려면 아래 문제 해결 단계를 따르세요.

오류를 해결하려면 다음 방법을 시도해 보세요.

컴퓨터 AndroidiPhone/iPad
더보기

페이지 새로고침

일반적으로 페이지를 새로고침하면 오류가 해결됩니다.

왼쪽 상단에서 새로고침 을 클릭합니다.

그래도 문제가 해결되지 않는 경우

1단계: 인터넷 연결 확인하기

컴퓨터가 Wi-Fi 또는 유선 네트워크에 연결되어 있는지 확인합니다.

오류가 발생한 탭을 새로고침해 봅니다.

2단계: 캐시 삭제하기

Chrome에 페이지 로드를 중지시키는 정보가 저장되었을 수 있습니다.

시크릿 창에서 페이지를 엽니다.

  1. 컴퓨터에서 Chrome을 엽니다.
  2. 오른쪽 상단에서 더보기 새 시크릿 창을 클릭합니다.
  3. 시크릿 창에서 페이지를 열어봅니다. 창이 열리면 캐시와 쿠키를 지우세요.

캐시 및 쿠키 삭제하기

  1. 오른쪽 상단에서 더보기 도구 더보기 인터넷 사용 기록 삭제를 클릭합니다.
  2. '기간' 옆의 전체 기간을 선택합니다.
  3. 캐시된 이미지 및 파일과 쿠키 및 기타 사이트 데이터를 선택합니다. 다른 유형의 데이터는 선택 취소합니다.
  4. 데이터 삭제를 클릭합니다.
  5. 오류가 발생한 탭을 새로고침해 봅니다.

3단계: 기타 탭, 확장 프로그램, 앱 닫기

기기 메모리가 부족하여 앱, 확장 프로그램, 프로그램을 실행하면서 사이트를 로드할 수 없는 경우도 있습니다.

  1. 메모리를 확보합니다.
    • 오류 메시지가 표시된 탭을 제외한 모든 탭을 닫습니다.
    • 실행 중인 다른 앱이나 프로그램을 종료하고 앱 또는 파일 다운로드를 일시중지합니다.
    • Chrome에서 필요 없는 확장 프로그램을 제거합니다. 오른쪽 상단에서 더보기 도구 더보기 확장 프로그램을 클릭합니다. 사용하지 않는 확장 프로그램에서 삭제를 클릭합니다.
  2. 오류가 발생한 탭을 새로고침해 봅니다.

아직도 문제가 해결되지 않았나요?

4단계: 컴퓨터 다시 시작하기

프로그램이나 앱이 페이지 로드를 중지하는 경우도 있습니다.

  1. 컴퓨터를 다시 시작합니다.
  2. 페이지를 새로고침해 봅니다.
5단계: Chrome 업데이트하기
  1. 컴퓨터에서 Chrome을 엽니다.
  2. 오른쪽 상단에서 더보기 설정을 클릭합니다.
  3. 왼쪽에서 메뉴 Chrome 정보를 클릭합니다.
  4. Chrome에서 사용 가능한 업데이트가 있는지 확인합니다. 확인된 내용을 보려면 몇 초 정도 기다려야 할 수 있습니다.
    • 업데이트가 있는 경우 다시 실행 옵션이 표시됩니다. 다시 실행한 후 오류가 발생한 탭을 새로고침해 보세요.
    • 안내가 없다면 최신 버전의 Chrome을 사용하고 있는 것입니다.
6단계: 원치 않는 소프트웨어 확인하기

Windows를 사용 중이라면 Chrome 정리도구를 사용하여 원치 않는 소프트웨어를 찾아서 삭제하세요.

Mac이나 Linux 컴퓨터를 사용 중이라면 원치 않는 소프트웨어 검색 및 제거에 관한 도움말을 참고하세요.

원치 않는 소프트웨어를 삭제한 후 오류가 발생한 탭을 새로고침해 보세요.

추가 지원

  • 다른 사이트는 정상적으로 열리지만 특정 사이트만 다운되는 경우, 사이트에 문제가 있을 수도 있습니다.
    • 새로고침해도 사이트가 열리지 않으면 의견 보내기를 클릭하여 Google에 알려 주세요.
    • Google에서 페이지 비정상 종료 문제를 더 빨리 해결할 수 있도록 자동 오류 보고를 사용 설정하거나 문제를 Chrome팀에 신고하세요.
    • 사이트 소유자에게 연락하여 문제를 신고하세요.
  • Chrome 도움말 포럼에서 전문가의 도움을 받으세요.
